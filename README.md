# assistant-lilo

Un plugin pour contrôler le potager connecté LILO Prêt à Pousser

## Configuration

```json
{
  "lilo": {
    "semaine": ["08", "00", "22", "00"],
    "weekend": ["12", "00", "23", "59"]
  }
}
```
